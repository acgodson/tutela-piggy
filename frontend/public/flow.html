Tutela

AI-Powered Livestock Health Monitoring System — Technical Architecture

Technology Stack

Frontend

  • React 19 (TypeScript)
  • Tailwind CSS 4
  • Drizzle ORM
  • PWA Ready

Backend

  • FastAPI (Python 3.10)
  • YOLOv8s (Ultralytics)
  • ByteTrack
  • OpenCV + NumPy

AI Services

  • Google Gemini 3.0 Flash
  • Custom YOLOv8 Model (23MB)
  • Instance Segmentation
  • Multi-Object Tracking

Infrastructure

  • Vercel (Frontend)
  • AWS EC2 (Backend)
  • Neon PostgreSQL
  • Docker Containers

Data Flow

1

Video Capture

V380 Pro IP camera streams RTSP video, or user uploads video file / uses browser camera. Frames extracted at configurable intervals.

2

Frame Processing

Web app extracts frames using Canvas API, converts to Base64, and sends to API route with expected pig count.

3

Object Detection

FastAPI backend runs YOLOv8s model for pig detection and instance segmentation. ByteTrack assigns persistent IDs across frames.

4

AI Validation

Gemini 3.0 Flash validates detections (confirming pigs vs. false positives) and reads painted identification numbers (1-30).

5

Alert Generation

System compares detected count vs. expected count, monitors movement patterns, and generates alerts for anomalies.

6

Data Persistence

All detections, tracking data, and alerts stored in PostgreSQL for analytics, historical review, and compliance reporting.

7

Visualization

Results rendered in real-time with bounding boxes, pig numbers, and alert indicators. Dashboard provides historical analytics and reports.
